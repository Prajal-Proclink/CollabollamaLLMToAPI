# Windows Scheduler Local API

This is a Python FastAPI web application designed to run locally.

## Setup Instructions

1. **Install Dependencies**:
   Open a terminal/command prompt in this folder and install the required libraries:
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the Application**:
   Start the FastAPI development server:
   ```bash
   python PythonTest.py
   ```

   The server will start at `http://127.0.0.1:8000`.

3. **Verify the Endpoint**:
   Open your browser or run curl to access the Windows-Scheduler-Local endpoint:
   - Endpoint URL: `http://127.0.0.1:8000/windows-scheduler-local`
   - Swagger Documentation: `http://127.0.0.1:8000/docs`
